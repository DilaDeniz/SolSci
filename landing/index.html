How it works

Three steps. One permanent proof.

From any research output to an on-chain certificate in seconds. Works with any file format — data, code, documents, images.

01

Hash locally

SHA-256 of your file is computed in the browser or CLI — entirely on your device. The file is never transmitted anywhere.

02

Register on-chain

A Program Derived Address (PDA) keyed by (wallet, hash) is created on Solana. It stores the hash, timestamp, and your metadata.

03

Verify anytime

Anyone with the original file can verify: hash it themselves, look up the PDA, compare. No account needed. No permissions required.

Features

Built for researchers,
not institutions

Every feature is designed to protect scientific integrity without adding friction to your workflow.

🔒

Privacy-first

Your raw data never leaves your machine. Only the 32-byte hash is stored on-chain. No server, no cloud upload, no data exposure.

⏱️

Tamper-proof timestamp

Solana's block time gives you a verifiable proof of existence. Once registered, the timestamp cannot be altered by anyone — not even us.

🤖

Local AI metadata (QVAC)

Powered by Tether's QVAC SDK, the AI assistant reads your file locally and suggests analysis type, tool, and description — completely offline.

⚡

CLI for CI/CD

The Rust engine integrates into any pipeline. Auto-stamp analysis outputs on every push. Works with GitHub Actions out of the box.

🌍

Any science, any file

Genomics, climate data, particle physics, economics — 28 research categories. PDF, CSV, FASTQ, code, images — any file format.

🔑

Self-sovereign

You own the record. Only you can close it and reclaim the rent. The PDA is keyed to your wallet — no one can register the same hash under your identity.

CLI Engine

Works in your terminal,
your CI, your pipeline

The Rust-based engine streams files in 64 KB chunks — no memory issues with multi-GB genomics files. Dry-run mode for testing without a transaction.

bash 
# Register a discovery
solsci-engine register \
  --file output.csv \
  --analysis-type climate_data \
  --tool-version "numpy/1.26.0" \
  --program-id 8cmvWB8S... \
  --keypair ~/.config/solana/id.json

→ Hashing output.csv (2.3 GB)…
→ SHA-256: a4f3c9e2d1b0...
→ PDA: FgZFsSyr...
→ Transaction confirmed: 3eTgTSHR...
✓ Discovery registered on Solana devnet

# Verify a record
solsci-engine verify \
  --hash a4f3c9e2d1b0... \
  --researcher 2rToMxB6... \
  --program-id 8cmvWB8S...
